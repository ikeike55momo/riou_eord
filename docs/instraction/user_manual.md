# キーワード自動提案Webアプリ - 使用方法ドキュメント

## 概要
このドキュメントでは、キーワード自動提案Webアプリケーションの使用方法について説明します。
このアプリケーションは、施設情報を入力し、AIを活用して3つのカテゴリ（メニュー・サービス、環境・設備、おすすめの利用シーン）のキーワードを自動生成するためのツールです。

## ログイン

1. ブラウザで以下のURLにアクセスします：
   - 本番環境: `https://keyword-suggestion-app.netlify.app`
   - 開発環境: `http://localhost:5173`

2. ログイン画面が表示されます。
   - メールアドレスとパスワードを入力します。
   - 「ログイン」ボタンをクリックします。

3. 認証に成功すると、ダッシュボードに遷移します。

## ダッシュボード

ダッシュボードでは、登録済みの施設一覧が表示されます。

### 主な機能
- 施設一覧の表示
- 新規施設の登録
- 既存施設の編集
- 施設の削除
- キーワード生成
- CSVエクスポート

## 施設情報の登録

1. ダッシュボードの「新規施設登録」ボタンをクリックします。
2. 施設情報入力フォームが表示されます。
3. 以下の情報を入力します：
   - 基本情報（必須）
     - 施設名
     - 施設タイプ
     - 住所
   - 連絡先情報
     - 電話番号
     - ウェブサイトURL
     - GoogleマップURL
   - 営業情報
     - 営業時間
     - 価格帯
     - 収容人数
     - 駐車場の有無
     - 予約必須かどうか
   - 評価情報
     - 平均評価
     - レビュー数
   - 詳細情報
     - 説明文
     - 特徴
     - 画像URL
     - メニュー項目
     - サービス
     - アクセシビリティ
     - 支払い方法
     - ソーシャルメディア
     - 認証
     - 対応言語
     - 近隣の観光スポット
     - 追加情報
4. 「保存」ボタンをクリックして施設情報を登録します。

## キーワードの生成

1. ダッシュボードで対象の施設の「キーワード生成」ボタンをクリックします。
2. キーワード生成画面が表示されます。
3. 「キーワードを生成」ボタンをクリックします。
4. AIが施設情報を分析し、以下の3カテゴリのキーワードを自動生成します：
   - メニュー・サービス
   - 環境・設備
   - おすすめの利用シーン
5. 生成されたキーワードが表示されます。

## キーワードの編集

1. 生成されたキーワードは編集可能です。
2. 各カテゴリのキーワードリストで、以下の操作が可能です：
   - キーワードの追加：「追加」ボタンをクリックし、新しいキーワードを入力
   - キーワードの編集：キーワードをクリックして編集
   - キーワードの削除：キーワードの横にある「×」ボタンをクリック
3. 編集が完了したら「保存」ボタンをクリックします。

## CSVエクスポート

1. ダッシュボードまたはキーワード編集画面で「CSVエクスポート」ボタンをクリックします。
2. CSVファイルがダウンロードされます。
3. CSVファイルには以下の情報が含まれます：
   - 施設ID
   - 施設名
   - 施設タイプ
   - 住所
   - メニュー・サービスキーワード
   - 環境・設備キーワード
   - おすすめの利用シーンキーワード

## 施設情報の編集

1. ダッシュボードで対象の施設の「編集」ボタンをクリックします。
2. 施設情報編集フォームが表示されます。
3. 必要な情報を更新します。
4. 「保存」ボタンをクリックして変更を保存します。

## 施設情報の削除

1. ダッシュボードで対象の施設の「削除」ボタンをクリックします。
2. 確認ダイアログが表示されます。
3. 「削除する」ボタンをクリックして施設情報を削除します。

## ログアウト

1. 画面右上のユーザーメニューをクリックします。
2. 「ログアウト」を選択します。
3. ログイン画面に戻ります。

## トラブルシューティング

### ログインできない場合
- メールアドレスとパスワードが正しいか確認してください。
- 管理者に連絡して、アカウントが有効かどうか確認してください。

### キーワード生成に時間がかかる場合
- 施設情報が複雑な場合、処理に時間がかかることがあります。
- ネットワーク接続を確認してください。
- しばらく待っても完了しない場合は、ページをリロードして再試行してください。

### CSVエクスポートができない場合
- キーワードが生成・保存されているか確認してください。
- ブラウザのダウンロード設定を確認してください。
